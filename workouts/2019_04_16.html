Clean and Jerk 
Squat Clean + 2 Push Jerk, 3 min
Find a heavy single for the day and accumulate a total of 5 heavy sets
Metcon (Time)
5-4-3-2-1
Hang Squat Clean (155/105)
Push Jerk (155/105)
Goal: Sub 6 min
RX+-205/145
Red - 135/95
White - 75/55

Loading on the barbell should be something that you can do a set of 5 hang squat clean in a row with as well as 5 push jerks in a row with.  The trick is going to be going back and forth between these two movements.  If you choose the weight right chances are you will be able to do the rep ranges from each set but it’s the rest  between movements that is going to be tough to make you want to get back on the barbell.  This is a short workout so keep the loading on the barbell something that allows for this.
Metcon (No Measure)
Death By Gymnastics Complex
1 Pull Up
1 CTB Pull Up
1 Bar Muscle Up
Goal: Complete the round of 5's and don't tear those hands!
Notes: Add one rep to each movement each minute till you can't complete the work within the min.
If you don’t have some of these movements try taking out the Bar MU and sticking with just the pull up and ctb.  If that’s the case, start by 2 and go up by 2’s each min 2/4/6/8, etc.
