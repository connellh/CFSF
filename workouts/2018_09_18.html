Been a while since we’ve done some higher volume work like this, especially in regards to pull ups so please be smart.  Scale back to fewer reps to avoid any undo soreness or if you are not used to this type of volume.  Breaking up reps early on will be the smart thing to do so you don’t hit muscular failure, especially on the pull ups and push ups.  This combo is a nasty one so be smart with it.  Taking reps back to 80’s, 60’s, or even 50’s if needed is fine.
