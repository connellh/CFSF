Metcon (Time)
30-20-10
Hang Power Clean and Jerk (135/95)
Calorie Row
Goal: Compare to previous time (7/2/2018) or Sub 14 min
Red - 95/65
White - 75/55

Revisiting and old friend.  We have done this workout in the past and it’s a great one to work on cycling a barbell with.  Be smart with your sets and break them up so you don’t go to failure.  Trying for sets of 5-7 is a great idea if you’re not sure where to start and the loading should allow you to do this.  Push the pace on the row since you have descending reps but make sure you are quick to get back on the barbell.  Lighten up the weight if needed in order to keep the barbell moving in bigger sets.
Metcon (Time)
10-9-8...1
Handstand Push Up
Strict Pull Up
Goal: Sub 14 min
Red - up to two mats or seated db z-press, band pull ups if needed
White - seated db z-press, banded pull ups

This is an old school .com classic that used to be programmed quite a bit.  Pull up variation is going to be tough so don’t stress quite as much on time today and focus on doing this one well.  Advantage is the descending reps but still be smart and break things up early so you avoid going to failure with either movement.  If you need to modify the pull up variation try doing them from a tuck position instead of an "L".  If you have HSPU but are developing capacity scale the reps first and keep the movement.  Otherwise you try one of the band/box subs or a "z" press.
