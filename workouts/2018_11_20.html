Starting off with some heavy work for the day.  Get some good build up sets in before jumping in on your work sets.  Weights can fluctuate as needed and make it a goal to have no failed reps while challenging yourself.

2 Rounds
50 Thrusters (95/65)
Run 1 Mile
Goal : Sub 25 min

Red- 75/55, 800 m Run

Be smart with this one, break it up, and dig in.  The thrusters should be a manageable weight that you can cycle efficiently so lighten up if needed.  Push the pace on the run but make sure you can get back on the bar and get to work when starting the second round.  Work to stay in the moment on this one and keep chipping away.
