Front Squat
15 mins to build up to a heavy set of 3 for the day
Metcon (No Measure)
EMOM 20 min
Odd: 1 Round of Dumbbell D.T. at (55's/35's)
Even: Run 200 meters, Row 12/8 cals, Ski 12/8 cals, Bike 12/8 cals
Goal: Just get it done
A mix of a light weight DB "D.T." and some cardio.  The DB’s should be light enough that with a push you are able to get the work done in roughly :45 sec or less, giving you some time to get ready for the cardio.  Same goes for the cardio.  You should be finishing with at least :10 sec or more to rest before starting the next round.  Lower the weight and shorten the distance/cals if needed in order to meet these parameters in the workout.
